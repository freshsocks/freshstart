Fresh Start
==============

A nodejs base app to start fresh from.

Includes
========

## Node Modules
* express
* jade
* stylus
* socket.io

## Bower Components
* d3
* gumby
* jquery
* modernizr
* underscore

Install
=======
```
git clone git://github.com/stevefloat/freshstart
```
```
cd freshstart
```
```
npm install
```
```
cd public
```
```
bower install
```
```
cd .. && node app
```
